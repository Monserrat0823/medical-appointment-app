# medical-appointment-app

## Objetivo general
Crear una aplicación web para la gestión y control de citas médicas, facilitando la organización de horarios para doctores y la reserva de turnos para pacientes de manera eficiente.

## Tecnologías a utilizar
* **Framework:** Laravel 11
* **Autenticación y Perfiles:** Laravel Jetstream
* **Componentes Dinámicos:** Livewire
* **Estilos:** Tailwind CSS
* **Lenguaje:** PHP 8.x
* **Base de Datos:** MySQL
